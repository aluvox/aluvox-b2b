# OEM Sampling Guide

## Overview

The sample stage is the quality gate in any OEM luggage sourcing relationship. Whatever is accepted at sample stage becomes the production standard for bulk orders. This guide covers the sampling process, timeline expectations, and evaluation criteria for brand buyers placing OEM luggage orders.

## Sampling Process Flow

```
Inquiry → NDA → Quotation → Sample Request
→ Sample Production → Sample Delivery → Evaluation
→ Written Approval / Revision Request → Bulk Production
```

## What to Specify in a Sample Request

Before requesting a sample, provide the factory with:

1. **Material specification**: Alloy grade (aluminum), resin brand/grade (PC/ABS), or material type
2. **Dimensions**: External dimensions, internal capacity requirements
3. **Color**: Pantone reference number or physical swatch
4. **Surface finish**: Brushed, sandblasted, anodized (aluminum) / matte, gloss, textured (PC/ABS)
5. **Hardware specification**: Lock type, wheel type, handle type, zipper brand
6. **Logo**: Artwork file, placement position, application method
7. **Interior**: Lining color, material, compartment configuration
8. **Quantity**: Expected bulk order quantity (affects pricing and MOQ confirmation)

## Sample Evaluation Checklist

When the sample arrives, evaluate against these criteria before approving for bulk production:

### Structural
- [ ] Shell dimensions match specification (measure at 3+ points)
- [ ] Wall thickness consistent across panels (measure at corners and centers)
- [ ] No visible deformation, warping, or surface irregularities
- [ ] Corner construction intact and reinforced

### Hardware
- [ ] Lock or latch engages and releases smoothly
- [ ] Operate lock/latch through 200 cycles — check for loosening
- [ ] Handle extends and retracts smoothly with load applied
- [ ] Wheels spin freely on multiple surfaces
- [ ] Zipper (if applicable) operates smoothly across full track

### Surface
- [ ] Color matches approved Pantone reference
- [ ] Surface finish consistent across all panels
- [ ] Logo placement, size, and quality match specification
- [ ] No scratches, blemishes, or finish inconsistencies

### Interior
- [ ] Lining color and material match specification
- [ ] Compartment dimensions and configuration match specification
- [ ] Dividers (if applicable) are secure and correctly positioned
- [ ] Mirror (if applicable) is securely mounted

## Critical Rule

**Document everything in writing.** Any deviation accepted at sample stage becomes the production standard. If you accept a sample with a 3mm dimension variance, your bulk order will have that same variance — or worse.

Issue written sample approval with:
- Photographs of the approved sample (all angles)
- List of any accepted deviations with tolerance ranges
- List of any required corrections before bulk production
- Signature and date

## Sample Fees

Sample fees vary by project and factory. Confirm with your factory:
- Is there a sample fee? If so, how much?
- Is the sample fee credited against the bulk order?
- How many sample iterations are included before additional charges?
