# MOQ Guide — Minimum Order Quantity

## Overview

MOQ (minimum order quantity) is the smallest number of units a factory will accept for a single production run. Understanding the cost logic behind MOQ helps buyers assess whether a factory's threshold is reasonable and structure orders to minimize capital exposure.

## Color Splitting Explained

Aluvox supports multi-color order splitting within a single style. Your total order quantity equals the MOQ — individual colors do not each need to reach the MOQ threshold.

**Example:**
- Silver: 100 pcs
- Black: 100 pcs
- Champagne gold: 100 pcs
- **Total: 300 pcs — MOQ met**

This structure allows brands to test color performance in market before committing to single-color volume on reorders.

## Four Factors That Affect MOQ

### 1. Material Type
Aluminum production carries higher fixed costs (stamping dies, anodizing) than PC or ABS (thermoforming). Despite this, Aluvox maintains a 300-piece MOQ across materials for existing tooling.

### 2. Customization Depth
- Existing style + color/logo change: MOQ lowest (300 pcs)
- Existing style + hardware modification: MOQ moderate
- New mold development: MOQ determined by project

### 3. Tooling Status
- Existing Aluvox tooling: 300 pcs standard
- New mold development: MOQ agreed per project based on tooling investment

### 4. SKU Count
Color splitting means MOQ applies to total order, not per color. Multi-SKU launches do not multiply the MOQ.
